# NewsPulse - Anime & Tech News Mobile App

A modern, high-fidelity mobile UI/UX prototype designed in Figma. This app provides a curated news experience focusing on **Anime, Technology, and Finance**, featuring a clean editorial layout and intuitive navigation.

## âœ¨ Features

### 1. Home Dashboard
* **Featured Hero Slider:** Large, immersive cover cards for top stories (e.g., Attack on Titan).
* **Category Chips:** Quick-tap filters for Anime, Tech, Finance, Arts, and Sports.
* **Modern Search:** Integrated search bar with community-themed placeholders.
* **Vertical Feed:** Clean list of the latest news with high-quality thumbnails.

### 2. News Detail View
* **Editorial Typography:** High-contrast serif headlines for a premium reading feel.
* **Author Metadata:** Clear attribution with "Published by" and date stamps.
* **Immersive Imagery:** Full-width header images for visual storytelling.

### 3. Navigation
* **Bottom Bar:** Quick access to Home, Favorites, and User Profile.
* **Smooth Transitions:** Smart Animate transitions between the feed and article views.
